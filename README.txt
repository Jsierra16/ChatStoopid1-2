README

Versión de Unity: 6000.0.36f1

Cambios y mejoras

El chat, que antes solo se mostraba en la consola, ahora se visualiza directamente en la escena.

Los mensajes enviados por el cliente se muestran en el servidor y viceversa.

Se permite enviar imágenes (ADVERTENCIA: no pueden superar los 17 KB exactos).

El audio sigue siendo unidireccional; si se envía un audio, la funcionalidad de chat deja de funcionar.

Instrucciones de uso
Opción 1: Full Editor

Abrir el proyecto con escenas aditivas, las escenas a utilizar son ClientTest y ServerTest.

En el lado derecho del botón de la webcam se encuentran los botones para iniciar y conectar el servidor.

Primero, hacer clic en Servidor y luego en Cliente.

Cuando aparezca el mensaje “Servidor iniciado y conectado”, ya se puede empezar a chatear.

Opción 2: Build + Editor

Abrir uno de los dos como Build (ubicado en la carpeta Builds) y el otro desde el Editor.

Asegurarse de abrir solo una escena en el Editor.

El funcionamiento de los botones es idéntico al de la Opción 1.

Opción 3: Builds

Si se dispone de dos computadoras en la misma red local, cada una puede abrir un Build para conectarse entre sí.
